macOS App Icon Generator (Web-Based)
一个纯前端实现的 macOS 应用图标生成工具，旨在帮助开发者快速将普通图片转换为符合 Apple Human Interface Guidelines (HIG) 规范的 .icns 图标文件。  

  A pure client-side web tool designed to help developers quickly convert standard images into professional .icns icon files that strictly follow the Apple Human Interface Guidelines (HIG).
